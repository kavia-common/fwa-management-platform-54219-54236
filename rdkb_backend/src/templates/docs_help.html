RDKB API Help

Welcome to the RDK-B Fixed Wireless Access platform API. Use the endpoints to authenticate, register devices, configure settings, and stream telemetry.

Authentication

Obtain a Bearer token using POST /auth/login with OAuth2 Password flow.

curl -X POST -d "username=user@example.com&password=pass" {{ base_url }}/auth/login

Devices

Register and manage devices using /devices endpoints.

Configuration

Upsert and fetch per-device settings via /configs.

Monitoring

Submit and query telemetry via /monitoring.

Base URL

{{ base_url }}

Authorization

Use the token returned from /auth/login as a Bearer token:

Authorization: Bearer <access_token>
For real-time features, integrate WebSocket notifications (not included in this starter) following similar auth patterns.
